# Tests Documentation

## 🚀 Quick Start

```bash
# Installer les dépendances
uv pip install pytest pytest-cov httpx

# Lancer tous les tests
uv run pytest tests/ -v

# Avec couverture
uv run pytest tests/ --cov -v
```
